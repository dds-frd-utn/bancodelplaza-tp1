Espia a tu pareja! YA YA YA

Para espiar a esa persona que tanto "amas" llama a la siguiente URL usando su número de Documento

/rest/clients/{DU}
La respuesta a esta llamada mostrara su informacion personal! En forma de JSON.

Por ejemplo, el link anterior devuelve el siguiente resultado:

Y que hay con las cuentas?

Registrate en un simple paso!

Para registrar tu propia cuenta para operar, debes llamar al metodo POST siguiendo el siguiente enlace: 

/rest/accounts/
enviando un JSON 
{
  "Titular": "Su ID personal"
}

La respuesta a esta llamada es un identificador del número de cuenta creada

Consultar Cuenta

Puede consultar cualquiera de sus cuentas o la de algun conocido usando la siguiente URL:

/rest/accounts/{ID}
enviando el ID de la cuenta que se desea obtener.

La respuesta es un JSON como el que sigue

{
  "casa":19
  "cbu":119
  "fechaAlta":"2019-08-23T03:00:00Z[UTC]"
  "idCliente":1
  "idCuenta":1
}
